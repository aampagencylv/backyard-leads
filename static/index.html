Email Signature Preview

This is what your signature looks like. Edit your profile fields above to update it.

API Keys

Org-level keys used by the prospector. Stored in the database (not the .env file), so you can rotate without SSH access. Empty value falls back to the server's environment variable.

Twilio Voice

Required for in-app calling. Find your credentials at console.twilio.com.

The fields below are needed for browser-based dialing (Phase 2). Skip for now if you're just buying numbers.

Deepgram

Telephony-grade transcription for recorded calls. Sign up at console.deepgram.com ($200 free credit, then $0.0043/min).

Blooio

iMessage messaging via Blooio. Sends from BMP's dedicated number to any iPhone. Higher response rates than SMS for B2B cold outreach. Blooio docs

HMAC verifies inbound webhooks. Get this from Blooio dashboard → Webhooks (shown once when you create a webhook). Without it, anyone who knows the URL can forge fake replies.

Messaging Tone & Direction

Org-wide strategic angle prepended to every AI generation (cold email, follow-ups, iMessage, post-call sequences). Default leads with AI findability / GEO / local SEO. Edit to refocus the team's voice — clear to revert to default.

Twilio Phone Numbers

Numbers cost $1.15/mo each. Assign one to each rep for personalized caller ID + better connect rates. Admin-only.

Calling

How do you want to dial? Browser uses your computer's mic + speaker. Phone Bridge rings your personal phone first and then bridges you to the prospect — works anywhere, no headset needed, great for offshore reps.

Audio Devices

Select which microphone and speakers your browser uses for calls. Saved per browser. Only applies to Browser dial mode.

Email Sending

When disabled, you can still generate and review sequences, but Send buttons are locked. Enable once your Resend domain is verified.
